# Task Management Fullstack

Fullstack task management project with:

- `backend/`: Node.js, Express, MongoDB API
- `frontend/`: React, Vite client

## Structure

```txt
task-management/
  backend/
  frontend/
```

## Run backend

```bash
cd backend
npm install
npm start
```

## Run frontend

```bash
cd frontend
npm install
npm run dev
```
